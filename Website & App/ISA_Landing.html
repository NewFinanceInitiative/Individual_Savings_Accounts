Instant Savings Accounts (Ether Based ISA's)

The Ether based ISA smart contracts offer unqiue alterantives for sovereign financial products that are supported with the consumption of blockchain technologies and digital currency to introduce an instance of applied decentralisation to custodianship in financial services and financial technology based products in the economy.

As individual consumables, the smart contract ISA can come in many different options and as they all offer one basic simple use case "the purpose of being consumed as a savings vehicle", how they are characterised with functionality then becomes an instance of economic activity that can be analysed with looking at the different types of ISA products are being consumed. The different types of smart contract ISA's available now from the New Finance Initiative are below.

1: Set Deposit Amount ISA

The type of ISA that can help with set financial goals and the application of regular savings.

2: Set Time Unlock ISA

This ISA comes with a strict set time limit and helps with target based personal finance.

3: Set Deposit, Set Time Unlock ISA

The standard for ISA's, to help to create financial infrastructure that can be dynamic and more consumable.
